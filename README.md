# Oraclia · Nucli del Sistema

Aquest repositori conté el prototip mínim del llenguatge simbòlic d’Oraclia:
un sistema de 21 símbols Unicode definits com a unitats estructurals per descriure processos.

Inclou:
- definició del nucli conceptual
- funcions dels símbols
- estructura formal del llenguatge
