Symbolic Self-Insight Prototype
This demo uses a fixed set of structural symbols to read internal movements in a situation. It does not interpret emotions, it does not predict, and it does not replace professional analysis.
The system looks for structural signals (uncertainty, decision point, limit, movement, change, etc.) using simple rules.

Use simple language: “I feel…”, “I’m trying to…”, “I don’t know if…”. The system will respond only with structural guidance, not personal judgement.
This prototype works with a closed set of 21 Unicode symbols defined by Oraclia. Here only a subset is used for demonstration purposes: ?, Y, ⊘, ◉, ∆, †, →, ≡, ◌.
